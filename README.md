## Overview

This is a simple web-based game inspired by cricket, similar to rock-paper-scissors. Players can choose between "Bat," "Ball," or "Stump" and play against the computer. It's a fun project developed to practice core web development skills.

## Live Demo

Experience the game directly in your browser:
 https://prerana-singh27.github.io/CricketGame/

## Features

-   Interactive gameplay: Choose Bat, Ball, or Stump.
-   Computer opponent: Play against an AI that makes random choices.
-   Score tracking: Keep an eye on your wins, losses, and ties.
-   Simple user interface: Clean and intuitive design built with basic web technologies.

## Technologies Used

-   **HTML:** For the basic structure and content of the game.
-   **CSS:** For styling the game, making it visually appealing.
-   **JavaScript:** For implementing the game logic, user interaction, and score management.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You only need a modern web browser to play this game.

### Installation

1.  **Clone the repository:**
    ```
    git clone https://github.com/Prerana-Singh27/CricketGame.git
    ```
2.  **Navigate into the project directory:**
    ```
    cd CricketGame
    ```

### How to Run

Open the `index.html` file in your preferred web browser.

Alternatively, if you have a Live Server extension (e.g., in VS Code), you can launch it for a development server experience.

## Usage

1.  Open the game in your browser.
2.  Click on one of the buttons: "Bat", "Ball", or "Stump".
3.  The computer will make its choice, and the winner of the round will be displayed.
4.  The score will update accordingly.
5.  Keep playing to see who wins the match!
